Recent Projects

Project 1

Semantic Safari

A definition-based game where the user is presented with a random definition and has to quickly select the correct word from a list of four synonyms. Planned on using Chat GPT to both pick definitions and generate synonyms, and the score is based on both speed and accuracy. We also created the project using Node.js and React.js. For fun, I also added a visual theme related to a safari.

Project 2

Connect 4 Manual

An engaging project crafting a beginner's guide to Java through a hands-on approach: building a Connect 4-inspired game. Seamlessly blending theory with practical application, the manual offers step-by-step instructions, code snippets, and clear explanations, empowering newcomers to Java programming while fostering a playful learning environment.
